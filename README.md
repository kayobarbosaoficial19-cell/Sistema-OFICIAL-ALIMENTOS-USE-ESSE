# MenuFácil

Sistema de cardápio digital para o Açaí da Lelê.

## Rotas

| URL | Descrição |
|-----|-----------|
| `/` | Loja — área do cliente |
| `/admin` | Painel administrativo (login obrigatório) |

## Desenvolvimento local

```bash
npm install
npm run dev
```

Acesse `http://localhost:3000`

## Deploy

Compatível com **Vercel**, **Netlify** e **GitHub Pages** (via export estático).

## Acesso Admin

Usuário: `admin`  
Senha: `1234`

> Altere as credenciais em `public/admin.html` antes de publicar em produção.
